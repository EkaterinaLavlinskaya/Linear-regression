# Linear-regression
# Анализ данных и предсказание выживаемости персонажей "Игры престолов"

## 📌 Бизнес-задача
Можно ли на основе характеристик персонажей (дом, пол, титул, семейные связи) предсказать, выживет ли он до конца саги? Проект демонстрирует полный цикл Data Science: от «сырых» данных до интерпретируемой модели.

## 📊 Данные
- **Источник:** Kaggle (датасет по вселенной "Игры престолов")
- **Размер:** 1500+ персонажей, 25+ признаков
- **Целевая переменная:** `isAlive` (1 — жив, 0 — мёртв)

## 🔍 Что сделано

### 1. Разведочный анализ (EDA)
- Визуализация распределений признаков
- Анализ пропусков (в некоторых признаках до 80% пропусков)
- Выявлены ключевые закономерности (например, связь между домом и выживаемостью)

### 2. Предобработка данных
- Созданы **флаги пропусков** (`_nan`) для всех ключевых признаков
- Группировка редких культур в осмысленные категории (`culture_grouped`)
- Обработка числовых признаков (заполнение пропусков)

### 3. Feature Engineering
- Создан бинарный признак `boolDeadRelations` — были ли у персонажа убитые родственники
- Добавлены флаги для всех ключевых категориальных признаков

### 4. Моделирование
- Обучена модель **логистической регрессии**
- Проведена балансировка классов
- Оценка качества на тестовой выборке

## 📈 Результаты
| Метрика | Значение |
|---------|----------|
| **Accuracy** | 0.80 |
| **Precision (класс 1)** | 0.80 |

**Матрица ошибок:**

**Инсайт:** Модель отлично находит живых персонажей (ошибка всего 0.5%), но хуже определяет умерших (находит 53%). Это связано с дисбалансом классов (умерших в 3.5 раза меньше).
